# HeadElf - Chief Software Architect Extension for Claude Code

A comprehensive Claude Code extension that provides world-class enterprise, application, and full-stack software architecture capabilities.

## Overview

HeadElf transforms Claude Code into a chief software architect with deep expertise across:

- **Enterprise Architecture**: Strategic technology planning, system integration, governance
- **Application Architecture**: Design patterns, microservices, distributed systems
- **Full-Stack Architecture**: Frontend, backend, data, infrastructure, DevOps

## Architecture

```
HeadElf/
├── skills/                    # Core architectural skills
│   ├── enterprise/           # Enterprise architecture capabilities
│   ├── application/          # Application design and patterns
│   ├── fullstack/           # Full-stack development architecture
│   └── governance/          # Architecture governance and standards
├── subagents/               # Specialized architectural subagents
│   ├── system-designer/     # Complex system design
│   ├── tech-evaluator/      # Technology assessment and selection
│   └── migration-planner/   # System migration and modernization
├── workflows/               # Architectural decision workflows
└── templates/              # Architecture documentation templates
```

## Skills Included

### Enterprise Architecture
- Strategic technology planning and roadmapping
- Enterprise integration patterns and governance
- Digital transformation strategies
- Technology portfolio management

### Application Architecture
- Microservices design and decomposition
- API design and management
- Event-driven architectures
- Design pattern selection and implementation

### Full-Stack Architecture
- Frontend architecture (SPA, micro-frontends, SSR/SSG)
- Backend architecture (REST, GraphQL, event sourcing)
- Data architecture (SQL, NoSQL, data lakes, streaming)
- Infrastructure architecture (cloud-native, containerization, serverless)
- DevOps and CI/CD pipeline design

## Installation

Place the HeadElf directory in your Claude Code skills directory and restart Claude Code.

## Usage

The skills are automatically discovered by Claude Code. Simply describe your architectural challenge or request architectural guidance, and the relevant skills will be invoked.
