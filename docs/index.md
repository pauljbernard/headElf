## Quick Start

### Try HeadElf
Experience C-suite executive intelligence with Claude Code:

```bash
# Install Claude Code (if not already installed)
npm install -g @anthropic/claude-code

# Clone HeadElf
git clone https://github.com/pauljbernard/headElf.git
cd headElf

# Install HeadElf skills in Claude Code
cp -r src/* ~/.claude/skills/headElf/

# Launch Claude Code with HeadElf
claude-code --enable-extension headElf
```
