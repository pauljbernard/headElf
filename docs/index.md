## Quick Start

### Prerequisites
- Claude Code installed and configured
- Node.js 16+ and Python 3.9+
- Git for version control

### Installation
```bash
# Clone the HeadElf repository
git clone https://github.com/pauljbernard/headElf.git

# Install in Claude Code skills directory
cp -r headElf ~/.claude/skills/

# Verify installation
claude-code --list-skills | grep headelf
```

### First Usage
```bash
# Launch Claude Code with HeadElf
claude-code

# Try your first executive consultation
> "I need a comprehensive M&A evaluation framework for a technology acquisition"
> "Design a global compliance strategy for our expansion into EMEA markets"
> "Develop a venture capital investment thesis for emerging AI companies"
```
